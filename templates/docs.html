To get started, here’s a brief overview of each field you’ll encounter:

TMDB API Key: Head over to The Movie Database to obtain your unique API key, which is essential for accessing their movie data.

Jellyfin URL: This is simply the URL of your Jellyfin server, where your media is hosted.

Jellyfin Token: You can find this token in your Jellyfin settings, which authenticates your requests.

Jellyseer URL: Enter the URL for your Jellyseer server, ensuring it's correct for seamless integration.

Jellyseer API Key: Generate this key in your Jellyseer settings to enable API access.

Jellyseer User: This field allows you to choose a Jellyseer user who will be making the requests to download movies and TV shows. Creating a dedicated user for this purpose can be vantaggioso.

Max Similar Movies & TV Shows: Here, you can specify a range from 0 to 20 to control the number of similar items you want to retrieve.

Cron Schedule: For scheduling, refer to Crontab Guru to understand how to format your cron jobs effectively.
